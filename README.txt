# gemini-chatbot
 Gemini based chatbot for mentoring and providing emotional and mental support.


# How to run
  1. Using Terminal Use cd path\to\the\app_file and run this command 'venv\Scripts\activate' 
  2. run this command 'python app.py' , a local host with address will be created go to the local host in your browser and use the program


# Note : Before running 'python app.py' make sure you have the following libraries installed,
		1. pip install Flask
		2. pip install google.generativeai
